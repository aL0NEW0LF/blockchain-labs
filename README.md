Developed on Ubuntu.

```shell
git clone https://github.com/aL0NEW0LF/blockchain-labs.git
```

```shell
cd blockchain-labs
```

```shell
mkdir obj
```

```shell
make
```

```shell
./main
```

To clean the project:

```shell
make clean
```
